1. const quotes = [] : สร้างกล่องว่างๆ เอาไว้เก็บคำคมทั้งหมด
2. Function: addQuote : ฟังก์ชันนี้ใช้เพิ่มคำคมใหม่เข้าไปในกล่อง
- เช็คก่อนว่า id เป็นตัวเลขมั้ย, content กับ author เป็น string รึเปล่า
- ถ้าใช่ก็ push เข้า array
- ถ้าไม่ใช่จะโยน error บอกว่าใส่มาผิด
3. Function: deleteQuote : ฟังก์ชันนี้ไว้ ลบคำคมออก ตาม id ที่บอกมา
- ใช้ findIndex หาตำแหน่งที่อยู่ของคำคมนั้น ถ้าเจอ splice จะทำการตัดทิ้ง ถ้าไม่เจอจะไม่ทำอะไรเลย
4. function updateQuote : ฟังก์ชันนี้ใช้ในการแก้ไขคำคม ตาม id ที่ระบุมา
- ใช้ find หา object ที่ id ตรงกัน
- ถ้าเจอจะทำการอัปเดตเฉพาะที่ส่งมา เช่นแก้ content หรือ author ก็ได้
5. function getAllQuotes() : ฟังก์ชันนี้ใช้นำคำคมทั้งหมดออกมาตรวจสอบ/ดู
